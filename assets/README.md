# Assets / Recursos

Este directorio contiene los recursos visuales de la landing page de LINK UP.

## Estructura recomendada

```
/assets
  /icons
    - savings.svg
    - network.svg
    - commission.svg
  /images
    - hero-image.jpg (opcional)
    - og-image.jpg (para redes sociales)
  /favicon
    - favicon.ico
    - apple-touch-icon.png
```

## Iconos incluidos en el HTML

Los iconos principales están incluidos como SVG inline en el HTML para mejor performance y personalización.

## Imágenes recomendadas

### Hero (opcional)
- Dimensiones: 1920x1080px
- Formato: JPG optimizado o WebP
- Peso máximo: 200KB

### Open Graph (para compartir en redes)
- Dimensiones: 1200x630px
- Formato: JPG o PNG
- Texto: Logo + mensaje clave de LINK UP

## Favicon

Genera un favicon con el logo de LINK UP usando herramientas como:
- https://realfavicongenerator.net/
- https://favicon.io/

## Optimización

Antes de usar imágenes en producción:
1. Comprimir con TinyPNG o Squoosh
2. Usar formato WebP cuando sea posible
3. Proporcionar versiones @2x para pantallas Retina
